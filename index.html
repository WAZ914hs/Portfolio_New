使用可能ツール
AviUtl

AviUtlをメインで使用し、プラグインやスクリプトを活用し
OP / ED / トランジションなど動きのある映像を制作しています。
無料ながら、After Effectsに匹敵する表現が可能です。

DaVinci Resolve

主に実況動画など、素材数の多い映像編集で使用しています。
カット編集・字幕入れ・音声調整など
全体の編集を効率よく進めることができます。

blender

無料で利用できる3D制作ソフトです。
オブジェクトのモデリングや
簡単なアニメーション制作に使用しています。
配信OPやMV用の素材など、
映像に必要な3D表現を自作する際に活用しています。

Visual Studio Code

Web制作時のエディタとして使用。
HTML / CSS / JavaScriptの
コーディング環境を整えるためのメインツールです

figma

Webデザイン・UI構築で使用。
配色・レイアウトなど、制作前の
設計ツールとして使用しています。

Photopea

Photoshop互換のオンライン編集ツール。
サムネイル制作や軽いレタッチで使用しています。
